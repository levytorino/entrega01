Surgimento do HTML

(Voltar)

O criador da HTML (Hiper Text Markup Language) chama-se Tim Berners-Lee em 1991.

Como funciona o html

(Voltar)

O HTML funciona através de códigos lidos e renderizados pelo navegador. O HTML se comunica com o navegador que interpreta o código e gera um conteúdo de facil assimilação para o usuário.

No HTML comunica tamanho das fontes, cor, links de outras páginas, de imagens ou ancoras na mesma página, entre outras coisas.

O que são Tags

(Voltar)

As tags comunicam o navegador a estrutura que está sendo construída. Uma tag pode informar um título, paragrafo, imafgem, link e etc.

Tags aprendidas até o momento

(Voltar)

A lista de tags a seguir foram aprendidas no curso do site dio.me e no curso complementar.

  • p: determinar o início e fim de um novo paragrafo.
  • ul: iniciar uma lista não numerada (não ordenada)
  • ol: iniciar uma lista ordenada por números em cada item da lista
  • li: delimita o inicio e fim de cada item da lista
  • a href: fazer referencia a algo; criar um link com um página, um link ancora.
  • blockquote: avança o alinhamento do paragrafo (pode usar para citação)
  • img src: linkar uma imagem da web ou de uma pasta
    inserindo width entre img e src pode-se altarear a largura da imagem: img width=500 src="link"
  • imput style:alterar a cor do texto
  • i: texto em italico
  • strong ou b: texto em negrito
  • h1,..., h6:Para texto de título, quanto maior o numero seguinte do h, menor o tamanho do título.
